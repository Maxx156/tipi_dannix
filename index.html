Демонстрация типов данных

Визуализация всех основных типов данных в JavaScript и правильной структуры программы с примерами использования и интерактивной демонстрацией.

Типы данных в JavaScript

Number (Числа)

Целые и дробные числа, а также специальные числовые значения.

let age = 25;
let price = 99.99;
let infinity = Infinity;

String (Строки)

Текст, заключенный в одинарные, двойные или обратные кавычки.

let name = "Анна";
let message = 'Привет!';
let template = `Hello ${name}`;

Boolean (Логический тип)

Принимает только два значения: true (истина) или false (ложь).

let isActive = true;
let isEmpty = false;
let result = 5 > 3; // true

Undefined

Означает, что переменная объявлена, но ей не присвоено значение.

let undefinedVar;
console.log(undefinedVar); // undefined

Null

Специальное значение, которое означает "ничего", "пусто" или "значение неизвестно".

let emptyValue = null;
let user = null; // пользователь не выбран

Object (Объекты)

Коллекция пар ключ-значение для хранения сложных структур данных.

let user = {
name: "Иван",
age: 30,
isAdmin: true
};

Array (Массивы)

Упорядоченная коллекция элементов с числовой индексацией.

let colors = ["красный", "зеленый", "синий"];
let numbers = [1, 2, 3, 4, 5];

Function (Функции)

Выполняемый блок кода, который можно вызывать многократно.

function sum(a, b) {
return a + b;
}
let result = sum(5, 3); // 8

Правильная структура программы

Хорошо структурированная программа должна содержать:

  • Импорт необходимых модулей (если требуется)
  • Объявление констант и переменных в начале
  • Определение функций с четкими названиями
  • Основную логику программы
  • Обработку ошибок и исключительных ситуаций
  • Чистый и читаемый код с комментариями
// 1. Объявление констант
const MAX_USERS = 100;
const APP_NAME = "DataTypes Demo";

// 2. Объявление переменных
let userCount = 0;
let isAppActive = true;

// 3. Определение функций
function addUser(userName, userAge) {
  if (userCount < MAX_USERS) {
    const newUser = {
      name: userName,
      age: userAge,
      id: Date.now()
    };
    userCount++;
    return newUser;
  } else {
    return null;
  }
}

// 4. Основная логика программы
function runApp() {
  console.log(`Запуск приложения: ${APP_NAME}`);

  // Создание пользователей
  const user1 = addUser("Алексей", 25);
  const user2 = addUser("Мария", 30);

  // Проверка результата
  if (user1 && user2) {
    console.log("Пользователи успешно созданы");
  }
}

// 5. Запуск приложения
runApp();
